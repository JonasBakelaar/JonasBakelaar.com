Emmetros Work Terms 1 and 2

Introduction

This post outlines my experience working at Emmetros Ltd in my first and second Coop work terms. This placement was my first coop experience, and provided me with many base industry skills that I can apply to future coop positions. This post will outline some of the tasks I completed throughout my terms at Emmetros as well as some of the skills I learned along the way.

Employer Info

Emmetros is a Waterloo based start-up focused on building software for people with Alzheimer's and memory loss. They do this through an iOS app called MemorySparx One and a web-based app that was developed in my time at the company called MemorySparx Connect.

Job Description

As a Quality Assurance Specialist, I was responsible for writing test cases and developing an automated testing suite to test the company's products. I learned the importance of establishing clear and consistent test cases when developing new software, and worked hands on with Java code and new (to me) frameworks to develop the automation testing suite. It also meant working with Emmetros' own API to complete automation tasks.

Approximately 3 months into the coop placement, I was also offered the opportunity to work as a Developer at the company and help develop their admin website. This meant working directly, hands-on with the company's backend to develop a tool that would let employee's access important site functionality without requiring the employee to learn how to use and write complicated database queries. This also involved learning about different frameworks such as Angular and PrimeNG to build a working UI for the administration website that synced with the backend.
